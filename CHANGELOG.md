# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-02-22

### 新增功能

#### 核心功能
- ✨ 渠道源倍率对比 - 对比多个中转商的倍率，找出最优价格
- 📊 当前倍率管理 - 查看和管理所有模型的倍率配置
- 💰 上游价格抓取 - 自动获取 OpenAI、Anthropic 等上游最新价格
- 🔍 价格对比更新 - 对比当前倍率与上游价格，智能调整
- 📈 价格历史记录 - 追踪价格变化趋势
- 📝 更新日志 - 记录所有倍率更新历史

#### 渠道管理
- 🔗 渠道源管理 - 添加和管理多个中转商
- 🔄 渠道对比 - 对比不同渠道的价格
- ✅ 自动签到 - 支持渠道源自动签到功能
- 🏥 活性检测 - 定期检测模型可用性

#### 用户体验
- 📱 现代化 UI - 基于 Ant Design 的美观界面
- 🎯 仪表盘 - 一目了然的数据概览
- 📖 使用说明 - 详细的操作指南
- 🔍 智能筛选 - 支持按状态、厂商、已拥有模型等筛选
- 🎨 状态标识 - 清晰的颜色标识（涨价/降价/新增/下架）

#### 技术特性
- 🐳 Docker 支持 - 一键部署
- 💾 SQLite 数据库 - 轻量级数据存储
- 🔒 安全性 - 敏感信息脱敏处理
- 📡 API 代理 - 安全的 API 请求转发
- ⚡ 性能优化 - 快速响应和数据处理

### 功能亮点

#### 渠道源倍率对比
- 支持同时对比多个渠道源
- 自动标识最低价格
- 显示输入/输出价格和倍率
- 支持"只看未设置倍率的模型"筛选
- 批量应用倍率，支持自定义加价比例

#### 智能价格对比
- 自动识别需要调整的模型
- 区分涨价、降价、新增、下架状态
- 批量应用建议倍率
- 支持按厂商、状态筛选
- 实时预览更新结果

#### 数据可视化
- 仪表盘展示关键指标
- 需要调整的模型数量提醒
- 签到统计和成功率
- 最近更新时间追踪

### 技术栈

- **前端**: React 18 + TypeScript + Ant Design + Vite
- **后端**: Node.js + Express + TypeScript
- **数据库**: SQLite
- **部署**: Docker + Docker Compose

### 已知问题

- 大量模型（>200个）时，更新操作可能较慢
- 某些渠道源的 API 可能返回 403 错误（已实现降级方案）

### 安全性

- API Key 存储在本地 SQLite 数据库
- 日志中敏感信息已脱敏
- 支持用户级别的权限控制

## 版本说明

### 版本号规则

- **主版本号（Major）**: 不兼容的 API 修改
- **次版本号（Minor）**: 向下兼容的功能性新增
- **修订号（Patch）**: 向下兼容的问题修正

### 更新类型

- `新增` - 新功能
- `变更` - 对现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - Bug 修复
- `安全` - 安全性修复
